# EVIDENCIA-COMPUTACION-EN-JAVA
 Esta es la fase final de la evidencia de Computación en java de el desarrollo de un programa para administración de citas.
## Instalación y configuración.
-Descarga y confirguracion del entorno de desarrollo de java.
-Descargar e instalación del JDK correspondiente.
-Descara del programa almacenado en el repositorio githhub.
## Uso del programa.
Este programa tiene la funcion de administrar citas en base a los pacientes, medicos y programar las citas.
